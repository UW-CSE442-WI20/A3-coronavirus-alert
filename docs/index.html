Coronavirus Map

This website presents a Coronavirus information map of mainland China. The map is colored so that the more severe the situation of an province is, the darker the red becomes. When hovering your mouse over the map, more detailed attributes of a province will be shown. There is a scrollable table that lists provinces in order of the numbers of confirmed cases. Note that this web page only includes the data from 01/23/2020 to 02/05/2020 EST.
